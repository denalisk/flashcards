Flashcards

JavaScript (JS)

is a high-level, dynamic, untyped, and interpreted programming language. It has been standardized in the ECMAScript language specification.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more. (eg. + - * / %)

Variables

a value that can change, depending on conditions or on information passed to the program. A container to place information in.

Variable naming conventions

  • camelCase for variable (and function) names
  • start with a letter
  • all one word, no spaces or dashes

Functions

a set of statements that performs a task or calculates a value.

Methods

actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

a value that is passed to a function upon execution.

Parameters

a placeholder given to a function declaration during defintion.

Return Statement

ceases execution in a function and returns a value to the caller.

Chaining Methods

syntax for invoking multiple method call in a single statement, without requiring variables to store the intermediate result.

Strings

stores a series of characters as written information, such as "John Doe." Primitive values created from literals.

Booleans

a binary variable, having two possible values called “true” and “false.”

Undefined

an undefined property that indicates a variable has not been assigned a value.

NaN

a "weirdo" Global Object that indicates a mathematical operation has failed, but still a number data type.

Escape Character

The \, used to turn special characters into string characters. Can also be used to insert other special characters into a string.

Alert

A basic javascript method that creates a pop-up with a message.

Comments

a programmer-readable explanation or annotation in the source code of a computer program.

JQuery

a fast and concise JavaScript Library that allows for easy utilization of advanced javascript functionality in a website.

Attributes

a modifier of an HTML element type that modifies the default functionality of an element type or provides functionality to certain element types unable to function correctly without them.
